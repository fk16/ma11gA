# Ableitungen  


1. **Polynomfunktionen**:
   - $f(x) = 2x^3 - 5x^2 + 3x - 1$
   - $g(x) = 4x^4 + 7x^3 - 2x^2 + 9$
   - $h(x) = x^5 - 3x^4 + 6x^2 - 8$
   - $p(x) = 3x^2 + 4x + 1$

2. **Exponentialfunktionen**:
   - $y(x) = exp(2x)$
   - $z(x) = 5\cdot exp(3x)$
   - $w(x) = exp(-x)$
   - $v(x) = 2\cdot exp(4x^2)$

3. **Logarithmusfunktionen**:
   - $l(x) = \ln(2x + 1)$
   - $m(x) = \ln(3x^2 + 5)$
   - $n(x) = \ln(4x - 2)$
   - $p(x) = \ln(x^2 + 3)$

4. **Trigonometrische Funktionen**:
   - $s(x) = \sin(2x)$
   - $c(x) = \cos(3x^2)$
   - $t(x) = \tan(4x)$
   - $a(x) = \sin(5x+1)$
  
5. **Gebrochen rationale Funktionen**:
   - $q(x)=x^{\frac12}+3$
   - $u(x)=\(x-4\)^{-\frac23}$
   - $k(x)=\sqrt{x^3-2x}$
   - $o(x)=4x^{0,4}+3x^{-0,2}$
